# Archery (Empowered)

![Empowered Archery](../assets/player-deck-back.webp){ width="340" align=right }

___
<p style="text-align: center;" markdown>[Ability](index.md)</p>
___
<p style="text-align: center;" markdown>:ongoing: Until the end of the next Combat round, all of your [Ranged units](../units/index.md#ranged-units) gain +1 :initiative: and +1 :attack: if the target is a non-adjacent [unit](../units/index.md).</p>
___


## Notes

- This ability is active during each ranged attack of the Combat round, which means that it applies to both attacks of [Elves](../units/elves.md) and [Marksmen](../units/marksmen.md).
