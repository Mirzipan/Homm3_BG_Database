# :resource_die: Resource Die

A Resource die is used to determine how much of which resource the player gains. Each side is unique and all 3 resources are represented equally, with each having a lower value and a higher value, where the higher value is double of the lower one.

| Sides |
| :---: |
| 3 :gold: |
| 6 :gold: |
| 2 :building_materials: |
| 4 :building_materials: |
| 1 :valuables: |
| 2 :valuables: |
